SOVEREIGN REPUBLIC

    Sovereign Republic
    2024 - PC - RolePlayGame - UE5 - 2 Mounths - 1 Worker
    Networking Development - C++ - BluePrints
    Pitch
    Inspired by the Star Wars Mod on Garry's Mod, this game let you play as a clone during the clone wars, it's mostly PVE and multiplayer. One of the main goal for me doing this was to learn more about C++ in Unreal Engine and Replication.
    Entity System
    The system is based on class inheritance, every interactible object is an entity so that I can do anything I want with it later down the road.
    So with this done I was able to get Items, Props, AI, Player etc as entities wich helped me to make gameplay related features and replication (like a replicated Inventory with RPC for example)
    Stats System
    It is replicated and it can be used on any actor because it's an Actor Component, it bind it self to the damage delegate, the base stats are set thanks to a Primary Data Asset.
    The good thing about this is that damage can be applied without adding anything else to this actor thanks to Unreal native damage system
