TEX ANIM

    Tex Anim
    2023 - Any Platform - Tooling - Unity - 1 Mounth - 1 Worker
    Development
    Pitch
    Tex Anim is a Unity Tool made with the purpose of making it easier to work in production with my animation system based on Textured Animations.
    Why ?
    To make a mobile game (Click to check it out) we needed a way to animate simple characters in great numbers.
    I ended up chosing the Textured Anmations way to handle animations because it would let me discover a bit about shader, mainly vertex shader (and a bit of compute shader too)
    Baking & Storing Textured Animations
    Using a Window Editor the tool would convert the animations on a selected character into a texture before saving it into a Custom Animations Bank (Scriptable Object).
    (How to Bake)
    Each row of the texture would be a frame of the animation, each column would be a vertex and like that each pixel would represent a vertex position at a precise frame (bottom images).
    How To Read Animations
    A Custom Animator had to be made to read the textures, it would handle all the transition between differents animations and transmit the commands to the GPU through a custom Vertex Shader
    Anything else ?
    A Graph Editor mimicking the Animator Window from Unity (animations settings, properties...) was made to be able to get more complex animations behaviour out of the Tex Anim system.
