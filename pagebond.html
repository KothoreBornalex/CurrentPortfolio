    BOND
    2023 - Top Down - SFML - 3 Weeks
    C++ - Engine Programming
    Pitch
    Bond is a students project made to learn C++ and the basics of making a game from ground zero. It Helped us getting a better understanding of how to organize our projects, how to work with CPP and how engines would work behind the wall. And we did learned A Lot !
    Level Editor
    The "Level Editor" gave us the possibility to create more complex levels and spend less time on the level creations since it was optimized to be easy to use. We saved and load levels inside/from JSON files.
    Entities System
    We needed a way to update objects, to list them, to destroy them... So we created a system for it, Entities are the back bone of everything it contains a lot of functinnalities going from handling the clearing of memory to the collision type of the entity.
    Lights & Shadows
    I made a raycast system based on the Entities System and I made it so that the outside of the shape created out of all the connections between all raycasts impacts would be the shape of the shadow.
    Then I just rendered the shadow under the buildings but above the AI and tada !
