    2023 - Mobile - Clicker - Unity - 3 Mounths - 2 Workers
    Optimization - Optimization - Gameplay
    Pitch
    Wandering Traveller is a students project made to learn how to make a mobile game on Unity. In this project we discovered how much of a limited hardware a phone is compared with a computer but we learned a lot of work around to get something looking good.
    How To Optimization
    First, A pool system to limit the garbage collector (Nearly 0 bytes of GC thanks to that).
    Secondly, a tool to play animation on the GPU (vertex shader...), after baking animation as a texture I saved it into a scriptable object built as an animations bank.
    Click To know more about: "Textured Animation Tool"
    Optimization Result
    Before the custom animation tool I got around 16-25 fps with 30 AI spawned and visibles on screen.
    With the custom animation tool the game fps Sky rocketed to 60 FPS with nothing less than 112 AI spawned and visibles on screen.
    Gameplay
    Stats system, 3C, AI and a few other things like Day/Night Cycle, game loop, wave spawner, settings etc
